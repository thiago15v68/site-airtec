# AIRTEC Site

Site institucional estatico preparado para publicacao no GitHub Pages.

## Publicacao no GitHub Pages

1. Crie um repositorio chamado `site-airtec`.
2. Envie todos os arquivos desta pasta para esse repositorio.
3. No GitHub, abra `Settings` > `Pages`.
4. Em `Source`, escolha `Deploy from a branch`.
5. Selecione a branch `main` e a pasta `/(root)`.
6. Salve e aguarde a URL ser gerada.

## Importante

- Se sua conta for `GitHub Free`, o repositorio provavelmente precisara ser `publico` para usar GitHub Pages.
- Se quiser manter o repositorio `privado`, o GitHub Docs informa que isso exige plano como `GitHub Pro` ou superior.

## Depois que a URL sair

1. Troque `YOUR_GITHUB_USERNAME` no arquivo `sitemap.xml`.
2. Acrescente a URL completa do sitemap no `robots.txt`, por exemplo:

   `Sitemap: https://SEU_USUARIO.github.io/site-airtec/sitemap.xml`

3. Envie novamente os arquivos atualizados.
4. Cadastre a URL no Google Search Console.
